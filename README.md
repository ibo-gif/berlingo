Приложение для мобильного обучения языкам с интерактивными уроками, как в Duolingo!

## 📱 Описание

**Berlingo** - это приложение для изучения иностранных языков на Android. Приложение содержит 8 интерактивных разделов для каждого языка с системой прогресса и опыта.

## ✨ Особенности

- 🎓 **8 уровней обучения** для каждого языка
- 🌍 **Поддержка 8 языков**: 
  - 🇬🇧 Английский
  - 🇨🇳 Китайский  
  - 🇰🇷 Корейский
  - 🇪🇸 Испанский
  - 🇫🇷 Французский
  - 🇩🇪 Немецкий
  - 🇯🇵 Японский
  - 🇷🇺 Русский

- 🎯 **Интерактивные вопросы** с объяснениями ответов
- 📊 **Система отслеживания прогресса** - видите свой уровень и опыт
- 🏆 **Награды за достижения** - +100 XP за каждый пройденный уровень
- 💾 **Автосохранение прогресса** - ваш прогресс сохраняется автоматически
- 🎨 **Современный интерфейс** на Material Design 3

## 🚀 Установка

### 1️⃣ Клонирование репозитория

```bash
git clone https://github.com/ibo-gif/berlingo.git
cd berlingo
```

### 2️⃣ Установка зависимостей

```bash
flutter pub get
```

### 3️⃣ Запуск на устройстве/эмуляторе

```bash
# Убедитесь, что у вас подключено устройство или запущен эмулятор
flutter run
```

### 4️⃣ Сборка APK (Release)

Есть несколько способов собрать APK:

#### Способ 1: Автоматически (рекомендуется)

**На Linux/Mac:**
```bash
chmod +x build_apk.sh
./build_apk.sh
```

**На Windows:**
```bash
build_apk.bat
```

Скрипт автоматически:
- ✅ Проверит установку Flutter
- ✅ Очистит кэш проекта
- ✅ Установит зависимости
- ✅ Соберет Release APK
- ✅ Покажет размер и путь файла

#### Способ 2: Вручную

```bash
# Сборка release APK
flutter build apk --release

# APK находится по пути: build/app/outputs/flutter-apk/app-release.apk
```

### 5️⃣ Сборка для Google Play (Bundle)

```bash
flutter build appbundle --release

# Output: build/app/outputs/bundle/release/app-release.aab
```

## 🎮 Как использовать приложение

### Главный экран
1. После запуска приложения видите все доступные языки
2. Нажимайте на флаги языков для выбора
3. Выбранные языки появляются в списке "Ваши курсы"
4. Нажимайте на языки в списке для начала обучения

### Экран разделов (Уровни)
- Каждый язык состоит из **8 разделов/уровней**
- Первый уровень сразу доступен
- Следующий уровень открывается после прохождения текущего
- Пройденные уровни отмечены зеленой галочкой ✓
- Заблокированные уровни отмечены замком 🔒

### Экран урока (Вопросы)
- Отвечайте на вопросы выбором правильного варианта
- И после выбора видите результат:
  - ✅ Правильный ответ - **зеленый**
  - ❌ Неправильный ответ - **красный**
- Каждый ответ содержит **объяснение**
- После всех вопросов видите итоговый результат
- **+100 опыта (XP)** за завершение уровня

### Система прогресса
- 📈 **Уровень**: показывает текущий уровень (1-8)
- ⭐ **Опыт**: накапливается за пройденные уровни
- 💾 Прогресс **автоматически сохраняется**

## 🔧 Разработка

### Добавление нового языка

Отредактируйте файл [lib/services/language_service.dart](lib/services/language_service.dart):

```dart
// В уже существующем списке LanguageData.languages добавьте:
Language(
  code: 'pt',                    // Код языка (ISO 639-1)
  name: 'Португальский',         // Название на русском
  flag: '🇵🇹',                  // Флаг страны
  description: 'Learn Portuguese', // Описание
),
```

Затем добавьте уроки в функцию `_generateLessonsFromTemplate()`:

```dart
'pt': [
  {
    'id': 1,
    'level': 1,
    'title': 'Приветствия',
    'description': 'Основные приветствия',
    'questions': [
      {
        'id': 1,
        'type': 'multiple',
        'question': 'Как сказать "Привет"?',
        'options': ['Olá', 'Adeus', 'Obrigado', 'Por favor'],
        'correctAnswer': 'Olá',
        'explanation': 'Olá - это португальское приветствие',
      },
      // ... еще вопросы
    ]
  },
  // ... еще 7 уровней (всего должно быть 8)
]
```
